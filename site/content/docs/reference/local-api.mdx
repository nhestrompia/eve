---
title: Local API
description: HTTP endpoints served by the trusted local eve runtime.
---

`eve dev` starts the local runtime on localhost. It serves the web UI, local API, and MCP endpoint.

## Endpoints

```text
GET  /api/config
GET  /api/repos
GET  /api/repos/{repoId}
POST /api/repos/{repoId}/open-editor
GET  /api/repos/{repoId}/snapshots
GET  /api/repos/{repoId}/snapshots/{snapshotId}
GET  /api/repos/{repoId}/snapshots/{snapshotId}/sessions
GET  /api/repos/{repoId}/snapshots/{snapshotId}/sessions/{sessionKey}
POST /api/repos/{repoId}/snapshots/{snapshotId}/checkout
POST /mcp
```

## Intended use

The local API supports the bundled eve UI and local development tooling. For agent integration, prefer MCP resources and tools because they expose product-history operations directly.

## Security

Keep the runtime bound to localhost. Do not expose the local API to untrusted networks or untrusted agent clients.
